# Todo-app frontend

This project builded with vue.js It's a most basic todo app.

## Project setup development with docker
```
docker-compose up
```

OR


```
docker build -t todoapp-frontend .
docker run -it -p 80:80 --rm todoapp-frontend
```

## Testing
```
npm run test
```
